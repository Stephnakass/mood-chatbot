# Mood Chatbot

## Overview
This project is a command-line Python chatbot that asks for the user's name, responds to mood inputs, and simulates conversation using typing effects, thinking pauses, and randomized responses.

## Features
- Collects user input through the command line
- Remembers the user's name during the session
- Responds to different moods with custom messages
- Uses random replies for more natural variation
- Includes a typewriter-style text effect
- Includes a looping conversation until the user exits

## How It Works
1. The program asks for the user's name
2. It repeatedly asks how the user is feeling
3. It checks the mood entered by the user
4. It selects an appropriate response
5. The conversation continues until the user types `exit`

## Example Output
```text
What is your name?
Stephanie
How are you feeling today, Stephanie?
tired
Thinking...
Stephanie, Maybe a nap is calling your name (￣o￣) zzZ
```

This project was built to practise core Python concepts including functions, loops, conditionals, string handling, user input, and basic user experience design.
